중앙대 다빈치 캠퍼스 맛집 & 카폐 공유 사이트

주제 선정 이유

1년동안 축적시켰던 내리 맛집 정보를 공유하고 싶어졌다.

중앙대 안성캠퍼스 학생들이 자주 이용할 수 있는 맛집 및 카페 정보를 한눈에 제공하여 편리함을 극대화하고 싶었다. 학교 주변 식당이나 카페를 찾는 데 걸리는 시간을 줄이고, 다양한 옵션을 비교할 수 있는 유익한 정보가 되었으면 하는 바람이다.

제작과정

1단계: 자료수집 및 정리

내가 가지고 있는 정보와 지인의 정보를 이용해서

2단계: 사이트 구성 설계

주변에 있는 사이트들과 템플릿들을 살펴보면서 전체적인 구성 설계

3단게: 제작

구성했던 내용대로 사이트 제작

제작

visual studio code 사용. free-css.com 에서 템플릿 이용. Netify.com 사용

느낀점

사이트 개발은 처음이라 어려운 점이 많았다. 하지만 원하는 내용으로 사이트를 만들어보는 기회를 가져 좋았다. 더 실력을 늘린 후에는 더 멋진 작품을 디자인하고 제작하고 싶다.

아쉬운 점

전체적인 디자인을 템플릿을 참고하고 사용해서 만든 것이 너무 아쉽다. 더 실력을 늘려서 전체적으로 모두 만들어보는 기회를 가져보고 싶다.
